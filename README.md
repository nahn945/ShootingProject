# ShootingProject


## コンセプト
短時間で爽快感を得られる、
弾幕処理と演出に重点を置いたシューティング。

## ゲーム概要
- 想定プレイ時間：1ステージ約5分
- ジャンル：2D縦スクロールシューティング
- プラットフォーム：PC(Windows, Mac)

## 操作方法
- 移動：上下左右
- ショット：Z
- ボム：X
- 低速化：LSHIFT

## 使用技術
- Visual C++
- Siv3D(C++フレームワーク)
- Jsonデータ管理
- CSVデータ管理

# コーディング・命名規則

## コーディング規則
- **変数名**：lowerCamelCase  
  例：`playerSpeed`, `currentScene`

- **構造体・クラス名**：UpperCamelCase  
  例：`Player`, `EnemyManager`

- **定数名**：SNAKE_CASE 
  例：`MAX_HP`, `DEFAULT_SPEED`

## 命名規則
- **ブランチ**
  `issue番号/feature or fix/ユーザー名/issue名`

- **プルリクエスト**
  `feat or fix: issue名`
